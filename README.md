## Contexte et objectifs

![Logo Bottleneck](P6_logo_Bottleneck.jpg) 
Ce projet consiste à nettoyer, fusionner et organiser différents jeux de données afin de reconstituer des informations cohérentes sur les ventes d’une boutique.

## Soft Skills
- Rigueur dans le traitement des données  
- Compréhension des sources d’erreur  
- Documentation claire des étapes de nettoyage  
- Analyse critique des incohérences  

## Compétences
- Détection des valeurs aberrantes  
- Traitement des doublons et valeurs manquantes  
- Fusion de datasets multi-sources  
- Analyses univariées et multivariées  

## Outils utilisés
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn
